Description détaillée

Gestionnaire de toutes les ressources utilisées dans le projet.

Gère et permet d'avoir accès à, de rajouter ou d'enlever des images, des textures, des modèles 3D et des polices de caractères associés à un nom.

Auteur
Antoine Legault, Jade St-Pierre Bouchard, Tai Chen Li, Samuel Labelle
Date
28 mars 2018
Version
0.2
Avertissement
Ajouter avec add() une ressource dont le nom est déjà utilisé écrasera l'ancienne ressource et causera une fuite de mémoire. removeResource() détruit la ressource en plus de libérer son identificateur.
