Description détaillée

Représentation de l'interface en jeu.

Interface pour construire et pour donner les informations sur la partie.

Auteur
Antoine Legault , Guillaume Julien - Desmarchais, Mickaël Grisé-Roy
Date
24 mars 2018
Version
0.7
Avertissement
Manque la boussole (pour indiquer le sens du vent) et le compte a rebours. Les labels ne s'ajuste pas selon la grosseur du nombre.
Bogue:
Problemes avec le sideWindow , fonctionne à 90%
